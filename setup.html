üìã Part A: Fork the Repository
  1. Go to github.com/jameskruck/claude-code-workshop
  2. Click the "Fork" button in the top-right
  3. Keep all default settings and click "Create fork"
  4. ‚úÖ You now have your own copy at github.com/your-username/claude-code-workshop
üìã Part B: Clone to Your Computer (Optional)

  3. Open your terminal/command prompt and navigate where you want the folder
  4. Run: git clone https://github.com/YOUR-USERNAME/claude-code-workshop.git
  5. ‚úÖ You now have the workshop files on your computer!

3

Deploy Your Workshop

5 minutes

Get your personal workshop running online:

  1. In Render dashboard, click "New +" ‚Üí "Web Service"
  2. Connect your GitHub account and select your forked repository
  3. Render auto-detects all settings - just click "Create Web Service"
  4. Wait 2-3 minutes for deployment (you'll see a live URL when done)
  5. Click the URL to test - you should see this same workshop interface
4

Enable AI Chat

3 minutes

Add your API key so the AI assistant can help you:

  1. In Anthropic console ‚Üí Settings ‚Üí API Keys
  2. Click "Create Key" and copy the key (starts with sk-ant-)
  3. In Render ‚Üí your deployed service ‚Üí Environment tab
  4. Add variable: ANTHROPIC_API_KEY = [paste your key]
  5. Click "Save Changes" (service will redeploy automatically)
